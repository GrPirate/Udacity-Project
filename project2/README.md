
# 街机游戏克隆项目

> 该项目是一个基于canvas的模仿青蛙过河的一个小游戏

# 技术栈
- canvas
- JavaScript面向对象编程（使用class创建Player和Enemy）

# 运行游戏
将项目下载下来，直接使用浏览器打开index.html即可

# 游戏玩法
1. 输入玩家昵称
2. 选择你喜欢的角色开始游戏
3. 使用使用上下左右操作角色躲过敌人到达对岸即可通关
4. 通关会记录玩家昵称和通关时间，并进入排行榜
5. 点击playAgain按钮可重新开始游戏
